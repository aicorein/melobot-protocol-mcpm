## 💬 简介

基于 [melobot](https://github.com/aicorein/melobot) 的通用 Minecraft 服务端进程管理协议。

可实现类似于 MCDR 的功能，但提供更多 melobot 支持下的新功能与新特性，目前仍在开发中。
